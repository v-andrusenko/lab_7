# Лабораторная 7

1) В WPF проекте создать возможность ввода массива размерностью N элементов, заполнить его произвольными целыми значениями.Динамически (если значение меняется в интерфейсе) выводить наибольшее значение массива, наименьшее значение массива, общую сумму элементов, среднее арифметическое всех элементов, вывести все нечетные значения. Реализовать вывод сообщения об ошибке в случае ошибочного ввода значений массива.

2) Создать класс MyMatrix, обеспечивающий представление матрицы произвольного размера с возможностью изменения числа строк и столбцов. Написать программу, которая выводит на экран матрицу и производные от нее матрицы разных порядков.

![task2](https://user-images.githubusercontent.com/70683676/121809472-9b0cc280-cc65-11eb-8979-c72578a75105.gif) 

3) В WPF проекте создать создать класс Article, содержащий следующие закрытые поля: название товара; название магазина, в котором продается товар; стоимость товара в гривнах. Создать класс Store, содержащий массив элементов типа Article. Обеспечить следующие возможности: добавление информации о товаре, вывод информации о товаре по номеру с помощью индекса; вывод на экран информации о товаре, название которого введено с клавиатуры, если таких товаров нет, выдать соответствующее сообщение;
